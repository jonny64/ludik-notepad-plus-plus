1. DESCRIPTION

    this simple plugin brings some of steludio functionality to notepad++:
     - function switching (Alt+G for get_item, Alt+M for draw_itemm, etc.)
     - model lookup (F6) for current screen

2. INSTALLATION

    provided installer should auto-detect appropriate directory for plugin unpacking
    manual installation: copy nppludi.dll to notepad++\plugins directory and restart it
